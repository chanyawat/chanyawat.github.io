    จงกำหนดสถานะเริ่มต้น (Initial state), การทดสอบเป้าหมาย (Goal test), Successor function และ Path cost function ให้กับปัญหาในข้อต่อไปนี้

    1. ปัญหาการลงสีในแผนที่โดยใช้สีเพียง 4 สี สมมุติว่าอาณาจักรแห่งหนึ่งประกอบด้วยแคว้นเล็กแคว้นน้อยจำนวนหนึ่ง ต้องการระบายสีในแผนที่ให้แคว้นที่อยู่ติดกันมี สีต่างกัน และสมมติว่าไม่มีแคว้นใดมีอาณาเขตอยู่ติดกันเกินกว่า 4 แคว้น
    2. ปัญหาลิงกับกล้วย มีลิงสูง 3 ฟุต อยู่ในห้องที่มีกล้วยแขวนติดเพดานสูง 8 ฟุต ลิงต้องการกินกล้วยหวีนี้ ในห้องมีกล่องสูง 3 ฟุตที่ซ้อนกันได้ เคลื่อนที่ได้ และปีนได้ จำนวน 2 กล่อง
